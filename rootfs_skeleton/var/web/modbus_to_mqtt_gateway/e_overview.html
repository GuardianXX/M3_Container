Contenedor de monitorización ModBus con publicador MQTT y registro de datos.

Este contenedor incluye software de terceros entre los cuales se encuentran:
  • Librería ModBus (libmodbus)
  • Publicador MQTT (Mosquitto)
  • Este programa registra periodicamente 1 registro de ModBus para posteriormente
  • enviar el valor del registro a un tema en un broker MQTT y al mismo tiempo guardar
  • un reporte llamado "Reporte de "FECHA_DEL_DÍA".txt, el cual puede ser descargado (aquí)
  • El esquema de funcionamiento es el siguiente:

Dispositivo
ModBus
→
Programa en SmartBox
→
Publicador
MQTT
→
Subscriptor
MQTT








Aplicación diseñada por SE3 basado en contenedor de INSYS.
Para poder funcionar se necesita:

  • Conexión a Internet
  • Dispositivo conectado al puerto ModBus
